## INSTALLATION FOR TESTING PURPOSES
This section provides instructions for setting up the development environment for testing purposes.

It is possible to set up a development environment using Kind and deploy DREEM on it to test the scaling actions.
This repo includes manifest to set-up a cluster with KinD and with Proxmox as infrastructure provider, but it is also possible to set-up a cluster on any other infrastructure provider and deploy DREEM on it.
For better testing, we suggest to set-up a cluster using Metal3 with the metal3-dev-env repository, which provides a ready-to-use environment for testing DREEM on bare metal servers.
